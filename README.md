# Calculadora en Python
Este proyecto es una calculadora básica desarrollada en Python que permite realizar operaciones matemáticas simples desde la terminal.

Funciones

- Suma
- Resta
- Multiplicación
- División
- Validación de datos ingresados
- Manejo de errores y excepciones

Tecnologías utilizadas

- Python 3

Cómo ejecutar el programa

1. Descarga o clona el repositorio.
2. Abre una terminal o CMD en la carpeta del proyecto.
3. Ejecuta el siguiente comando:

python calculadora.py

O en algunos sistemas Windows:

py calculadora.py

Objetivo del proyecto

Este proyecto fue creado con fines educativos para practicar:

- Funciones en Python
- Ciclos "while"
- Condicionales "if/elif"
- Manejo de excepciones ("try-except")
- Entrada y salida de datos
